# AI-Powered Real-Time Monitoring Dashboard

## Overview
Real-time monitoring dashboard built at Idelji. AI-powered log analysis with AWS Bedrock.

## Tech Stack
- Backend: FastAPI, Python, WebSocket
- Frontend: React, TypeScript
- AI: AWS Bedrock, LLM
- Infrastructure: Kubernetes, Docker

## Key Achievements
- 1M+ events per minute processing
- 35% reduction in incident response time
- 20+ enterprise clients adopted

## Project Structure
- k8s/ - Kubernetes manifests
- vscode/ - IDE configuration
- requirements.txt - Python dependencies
- Makefile - Build automation
